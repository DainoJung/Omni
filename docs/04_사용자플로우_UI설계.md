### 2.1 STEP 1: 입력 데이터

#### 입력 필드 정의

| 필드 | 타입 | 필수 | 설명 | 예시 |
|------|------|------|------|------|
| `brand_name` | text | ✅ | 브랜드명 | "나이키", "샤넬", "다이슨" |
| `description` | textarea | ✅ | 행사/상품 설명 | "2026 봄 신상품 런칭..." |
| `images` | file[] | ⬜ | 상품 이미지 (복수 가능) | PNG, JPG, 5MB 이하 |
| `event_period` | date_range | ⬜ | 행사 기간 | 2026.02.01 ~ 02.28 |
| `price_info` | text | ⬜ | 가격/할인 정보 | "정가 50,000원 → 39,000원" |
| `category` | select | ⬜ | 카테고리 | 패션, 가전, 식품, 뷰티 |

### 2.2.5 STEP 2.5: 스타일 선택 (컬러 + 톤앤매너)

> 🎨 레이아웃 선택 후, **컬러 프리셋**과 **톤앤매너**를 설정합니다.

#### 컬러 프리셋 (5종)

| 프리셋 ID | 이름 | Primary | Secondary | Accent | 권장 카테고리 |
|-----------|------|---------|-----------|--------|--------------|
| `COLOR_BLACK` | 모던 블랙 | `#1A1A1A` | `#F5F5F5` | `#E53935` | 가전, 패션 |
| `COLOR_NATURAL` | 내추럴 | `#8B7355` | `#F5F0E8` | `#4A7C59` | 식품, 뷰티 |
| `COLOR_GOLD` | 프리미엄 골드 | `#C9A962` | `#1A1A1A` | `#FFFFFF` | 명품, VIP |
| `COLOR_FRESH` | 프레시 | `#00A86B` | `#FFFFFF` | `#FF6B35` | 식품, 건강 |
| `COLOR_ROMANTIC` | 로맨틱 | `#E8A0BF` | `#FFF5F8` | `#9B2335` | 뷰티, 패션 |

#### 톤앤매너 (AI 추천 + 직접 입력)

AI가 브랜드명과 설명을 분석하여 **3가지 톤앤매너**를 추천합니다:

## 5. PoC 범위 조정

### 5.1 기존 계획 vs 수정된 계획

| 항목 | 기존 | 수정 |
|------|------|------|
| **출력 유형** | 샤넬 30×30, DID 1080×1480 | 스크롤형 상세페이지 (860×가변) |
| **AI 기능** | 텍스트 검증 | 텍스트 생성 + 이미지 생성 |
| **UI** | 간단 입력 → 생성 | 입력 → 템플릿 선택 → 편집 → 출력 |
| **편집** | 없음 | 텍스트 수정 + 이미지 교체 |

### 5.2 PoC에서 구현할 것

✅ **포함**:
- 입력 폼 (브랜드명, 설명, 이미지)
- 템플릿 2종 (식품, 패션)
- Claude 텍스트 생성
- 이미지 생성 AI 연동
- 기본 편집 (텍스트 수정, 이미지 교체)
- PNG/JPG 출력

❌ **제외**:
- 고급 편집 (드래그앤드롭, 요소 추가)
- 템플릿 8종 전체
- 협업 기능
- 버전 관리
