## 1. PoC 개요

### 1.1 목적

사용자가 **브랜드명, 설명, 이미지**를 입력하면 AI가 **스크롤형 상세페이지를 자동 생성**하고, **편집 후 이미지로 출력**하는 시스템의 기술적 타당성 검증

### 1.2 성공 기준

| 항목 | 기준 | 측정 방법 |
|------|------|-----------|
| **자동화 데모** | 식품/패션 2종 템플릿으로 상세페이지 자동 생성 | 실제 데이터로 시연 |
| **생성 품질** | 수작업 대비 80% 이상 품질 | 현업 담당자 평가 |
| **전체 생성 시간** | 30초 이내 | 입력 완료 → 편집 화면 표시 |
| **편집 기능** | 텍스트 수정 + 이미지 교체 가능 | 기능 테스트 |
| **이미지 출력** | PNG/JPG 정상 출력 | 출력 테스트 |

### 1.3 범위

#### ✅ 포함 (In Scope)

| 항목 | 상세 |
|------|------|
| **출력물** | 스크롤형 상세페이지 (860px × 가변 높이) |
| **템플릿** | 식품(TPL_FOOD), 패션(TPL_FASHION) 2종 (레이아웃만 정의) |
| **컬러 프리셋** | 5종 (모던 블랙, 내추럴, 프리미엄 골드, 프레시, 로맨틱) |
| **톤앤매너** | AI 추천 3개 + 직접 입력 옵션 |
| **입력** | 브랜드명, 설명, 이미지 업로드 (수동 입력) |
| **AI 생성** | Claude (텍스트/톤앤매너 추천) + Nano Banana Pro (이미지) |
| **편집** | 텍스트 수정, 이미지 교체 |
| **출력** | PNG/JPG 이미지 다운로드 |
| **기술 스택** | Next.js + FastAPI + Supabase |

#### ❌ 제외 (Out of Scope)

- 이메일 자동 수집 (RPA)
- 기존 POP 유형 (샤넬 30×30, DID 등)
- 템플릿 5종 이상
- 고급 편집 (드래그앤드롭, 요소 추가/삭제)
- 사용자 인증/권한 관리
- 프로덕션 배포

```

   ├── "뷰티/패션" 카테고리 추천 표시
   ├── "패션 룩북" 레이아웃 템플릿 선택
   └── [다음: 스타일 선택] 버튼 클릭

2.5️⃣ 스타일 선택 (30초) ★ 신규
   ├── 컬러 프리셋 5종 표시 → "로맨틱" 선택
   ├── AI 톤앤매너 추천 3개 자동 표시
   │   └── "고급스럽고 우아한" 선택

   ├── 로딩 화면 (톤앤매너 반영 텍스트 생성 → 이미지 생성)
   └── 생성 완료 → 편집 화면 자동 이동

4️⃣ 편집 (60초)
   ├── 메인 카피 클릭 → 텍스트 수정 시연
   ├── 배너 이미지 클릭 → 이미지 교체 시연
   └── 미리보기 확인

5️⃣ 출력 (20초)
   ├── [내보내기] 버튼 클릭
   ├── PNG 고화질 선택
   └── 다운로드 완료 → 파일 확인
```
