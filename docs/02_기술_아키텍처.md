## 1. 시스템 개요

### 1.1 서비스 개념

사용자가 **폼 입력**으로 상품 정보를 제공하면, AI가 **동적으로 레이아웃을 생성**하고 **텍스트를 코드로 오버레이**하여 **웹 상세페이지를 즉시 생성**하는 시스템

> **핵심 원칙**: 레이아웃 동적 생성 + 텍스트 정확도 확보

### 1.2 핵심 변화 (v4.0)

| 항목 | 이전 (v3.0) | 현재 (v4.0) |
|------|-------------|-------------|
| **핵심 가치** | AI 텍스트 생성 | **동적 레이아웃 생성** |
| **텍스트 처리** | AI가 생성 | **코드로 오버레이** (정확도 확보) |
| **AI 역할** | 카피라이팅 | **비주얼 레이아웃 생성** |
| **출력 형태** | 이미지 파일 | **웹 기반** (HTML/CSS) |

## 5. 서비스 구현

### 5.1 AI 파이프라인 서비스

```python
# services/layout_pipeline_service.py

from google import genai
from google.genai import types
from PIL import Image
from io import BytesIO
import json
from typing import List, Dict

class LayoutPipelineService:
    """동적 레이아웃 생성 파이프라인"""

    def __init__(self, api_key: str):
        self.client = genai.Client(api_key=api_key)
        self.model_pro = "gemini-3-pro-image-preview"  # Nano Banana Pro
        self.model_vision = "gemini-2.5-pro"           # Vision 분석용

    async def analyze_input_and_select_prompt(
        self,
        products: List[Dict],
        brand_name: str,
        category: str
    ) -> str:
        """[단계 2] 입력 분석 → 적합한 섹션 프롬프트 선택"""

        analysis_prompt = f"""
상품 정보를 분석하고 가장 적합한 레이아웃 섹션을 선택해줘.

입력:
- 상품 개수: {len(products)}
- 브랜드: {brand_name}
- 카테고리: {category}
- 상품들: {json.dumps(products, ensure_ascii=False)}

사용 가능한 섹션:
- single_feature: 상품 1개 강조
- grid_2: 상품 2개 나란히
- grid_3: 상품 3개 그리드
- hero_banner: 캠페인 배너

JSON으로 응답:
{{"selected_section": "섹션명", "reasoning": "선택 이유"}}
"""

        response = self.client.models.generate_content(
            model=self.model_vision,
            contents=[analysis_prompt]
        )

        return json.loads(response.text)

    async def generate_layout_image(
        self,
        section_prompt: str,
        product_images: List[bytes],
        aspect_ratio: str = "3:4"
    ) -> bytes:
        """[단계 3] Nano Banana Pro로 레이아웃 이미지 생성"""

        images = [Image.open(BytesIO(img)) for img in product_images]

        response = self.client.models.generate_content(
            model=self.model_pro,
            contents=[section_prompt] + images,
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE'],
                output_options=types.OutputOptions(
                    image_config=types.ImageConfig(
                        output_width=2048,
                        aspect_ratio=aspect_ratio
                    )
                )
            )
        )

        for part in response.candidates[0].content.parts:
            if part.inline_data:
                return part.inline_data.data

        return None

    async def extract_text_positions(
        self,
        layout_image: bytes
    ) -> Dict:
        """[단계 4] Gemini Vision으로 텍스트 영역 좌표 추출"""

        img = Image.open(BytesIO(layout_image))

        response = self.client.models.generate_content(
            model=self.model_vision,
            contents=[TEXT_ANALYSIS_PROMPT, img]
        )

        return json.loads(response.text)

    async def run_pipeline(
        self,
        products: List[Dict],
        product_images: List[bytes],
        brand_name: str,
        category: str
    ) -> Dict:
        """전체 파이프라인 실행"""

        # 1. 입력 분석 + 섹션 선택
        section_result = await self.analyze_input_and_select_prompt(
            products, brand_name, category
        )
        selected_section = section_result["selected_section"]

        # 2. 레이아웃 이미지 생성
        section_prompt = SECTION_PROMPTS[selected_section]
        layout_image = await self.generate_layout_image(
            section_prompt, product_images
        )

        # 3. 텍스트 영역 좌표 추출
        text_positions = await self.extract_text_positions(layout_image)

        return {
            "section_used": selected_section,
            "layout_image": layout_image,
            "text_positions": text_positions,
            "products": products
        }
```

### 5.2 텍스트 오버레이 (프론트엔드)

```tsx
// components/TextOverlayRenderer.tsx

interface TextArea {
  id: number;
  position: string;
  bounds: {
    x: number;
    y: number;
    width: number;
    height: number;
  };
  recommended_font_color: string;
  suitable_for: string;
}

interface Props {
  layoutImageUrl: string;
  textAreas: TextArea[];
  products: Product[];
  brandName: string;
}

export function TextOverlayRenderer({
  layoutImageUrl,
  textAreas,
  products,
  brandName
}: Props) {
  return (
    <div className="relative w-full h-full">
      {/* 배경 레이아웃 이미지 */}
      <img
        src={layoutImageUrl}
        alt="Layout"
        className="w-full h-full object-cover"
      />

      {/* 텍스트 오버레이 레이어 */}
      {textAreas.map((area, index) => (
        <div
          key={area.id}
          className="absolute flex items-center justify-center"
          style={{
            left: `${area.bounds.x}%`,
            top: `${area.bounds.y}%`,
            width: `${area.bounds.width}%`,
            height: `${area.bounds.height}%`,
            color: area.recommended_font_color,
          }}
        >
          {area.suitable_for === 'headline' && (
            <span className="text-2xl font-bold">{brandName}</span>
          )}
          {area.suitable_for === 'subtext' && products[index] && (
            <span className="text-lg">{products[index].name}</span>
          )}
          {area.suitable_for === 'label' && products[index] && (
            <span className="text-base">{products[index].price}</span>
          )}
        </div>
      ))}
    </div>
  );
}
```
