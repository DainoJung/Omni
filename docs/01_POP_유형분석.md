```

│  폰트명              │  용도           │  특징          │

│  신세계 반부리       │  타이틀/헤더    │  강조, 세리프  │
```

## 5. 자동화 분석

### 5.1 자동화 가능 영역

| 영역 | 자동화 수준 | 구현 방안 | 우선순위 |
|------|------------|-----------|----------|
| **템플릿 선택** | ✅ 완전 자동화 | 사이즈+용도 기반 룰엔진 | P0 |
| **폰트/사이즈 적용** | ✅ 완전 자동화 | 폰트 룩업 테이블 | P0 |
| **텍스트 배치** | ✅ 완전 자동화 | 고정 좌표 + 동적 크기 조절 | P0 |
| **이미지 배치** | ⚠️ 반자동화 | Vision AI 누끼 추출 + 중앙 배치 | P1 |
| **색상/배경** | ⚠️ 반자동화 | 브랜드별 컬러팔레트 DB | P1 |
| **검수 (오탈자)** | ✅ 완전 자동화 | LLM 기반 텍스트 검증 | P0 |
| **검수 (이미지)** | ⚠️ 반자동화 | Vision AI 매칭 검증 | P1 |

### 5.2 휴먼 인더 루프 (Human-in-the-Loop) 필요 영역

1. **최종 검수 승인** - 생성된 POP 최종 확인
2. **예외 케이스 처리** - 템플릿에 맞지 않는 특수 요청
3. **브랜드 가이드 위반 판단** - AI가 판단하기 어려운 디자인 품질

### 5.3 데이터 요구사항

자동화를 위해 필요한 입력 데이터:

```yaml
필수_데이터:
  - 브랜드명: string
  - 행사명: string
  - 행사기간: { start: date, end: date }
  - POP_유형: enum [DID, LCD, 사이니지, 샤넬, B5, A5, ...]
  - 상품_이미지: image_url | file

선택_데이터:
  - 가격: number
  - 할인율: percentage
  - 프로모션_내용: text
  - 문의처: string
  - 브랜드_컬러: hex_code
```

## 6. 템플릿 매핑 로직 (예시)

```python
def select_template(request):
    """
    입력 요청 기반 템플릿 자동 선택
    """
    size = request.get('size')
    category = request.get('category')
    purpose = request.get('purpose')

    # 사이즈 기반 1차 분류
    template_map = {
        '1080x1480': 'DID',
        '1920x1080': 'LCD',
        '270x480': 'LOUNGE_SIGNAGE',
        '30x30': 'CHANEL',
        '170x240': 'B5',
        '140x200': 'A5',
        '115x255': 'LOUNGE_MENU'
    }

    base_template = template_map.get(size)

    # 용도 기반 2차 분류
    if base_template == 'CHANEL':
        if 'FIT' in purpose:
            return 'FIT_CHANEL'
        elif category == '가전':
            return 'CHANEL_ELECTRONICS'

    if base_template == 'A5':
        if purpose == '행사':
            return 'A5_EVENT'
        elif purpose == '안내':
            return 'A5_NOTICE'

    return base_template
```
