### 1.2 타이포그래피

#### 대체 폰트 매핑

| 원본 (신세계 전용) | 대체 폰트 | 비고 |
|-------------------|-----------|------|
| 신세계 반부리 (세리프, 타이틀) | **Noto Serif KR** | Google Fonts, 고급스러운 세리프 |
| 신세계 민부리 (산세리프, 본문) | **Noto Sans KR** | Google Fonts, 깔끔한 산세리프 |

> 추후 신세계 전용 폰트 확보 시 CSS 변수만 교체하면 됩니다.

#### 타이포그래피 스케일

```css
/* 관리자 UI */
--font-heading-1: 700 28px/1.3 'Noto Sans KR';    /* 페이지 타이틀 */
--font-heading-2: 600 22px/1.4 'Noto Sans KR';    /* 섹션 타이틀 */
--font-heading-3: 600 18px/1.4 'Noto Sans KR';    /* 서브 타이틀 */
--font-body: 400 15px/1.6 'Noto Sans KR';         /* 본문 */
--font-body-sm: 400 13px/1.5 'Noto Sans KR';      /* 작은 본문 */
--font-label: 500 13px/1.4 'Noto Sans KR';        /* 라벨 */
--font-caption: 400 12px/1.4 'Noto Sans KR';      /* 캡션, 도움말 */

/* 템플릿 렌더링 (식품/패션 공통) */
--tpl-font-title: 700 48px/1.2 'Noto Serif KR';   /* 메인 카피 */
--tpl-font-subtitle: 400 24px/1.4 'Noto Serif KR'; /* 서브 카피 */
--tpl-font-body: 400 16px/1.6 'Noto Sans KR';     /* 본문 */
--tpl-font-price: 700 32px/1.2 'Noto Sans KR';    /* 가격 */
--tpl-font-caption: 400 13px/1.4 'Noto Sans KR';  /* 캡션/유의사항 */
--tpl-font-cta: 600 18px/1.0 'Noto Sans KR';      /* CTA 버튼 */
```
