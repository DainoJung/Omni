## 1. 시스템 개요

### 1.1 서비스 개념

사용자가 **브랜드명, 설명, 이미지**를 입력하면 AI가 **상세페이지를 자동 생성**하고, 사용자가 **편집 후 최종 출력**하는 시스템

### 1.2 시스템 아키텍처


## 2. 기술 스택

### 2.1 핵심 기술

| 구분 | 기술 | 용도 | 선정 이유 |
|------|------|------|-----------|
| **Frontend** | Next.js 14 | 웹 애플리케이션 | SSR, App Router, 빠른 개발 |
| **Backend** | FastAPI | API 서버 | 비동기, 타입 힌트, 자동 문서화 |
| **Database** | Supabase | DB + Storage + Auth | PostgreSQL 기반, 실시간, 무료 티어 |
| **LLM** | Gemini API | 텍스트 생성 | 고품질 한국어 생성, 이미지 생성 통합 |
| **이미지 AI** | Nano Banana Pro | 이미지 생성/편집/합성 | Gemini 3 Pro 기반, 4K 지원, 다중 이미지 참조 |

```sql
-- 프로젝트 테이블
CREATE TABLE projects (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    status VARCHAR(20) DEFAULT 'draft',  -- draft, generating, editing, completed

    -- 입력 데이터
    brand_name VARCHAR(100) NOT NULL,
    description TEXT NOT NULL,
    category VARCHAR(50),
    event_period_start DATE,
    event_period_end DATE,
    price_info VARCHAR(200),

    -- 템플릿 (레이아웃)
    template_id UUID REFERENCES templates(id),

    -- 스타일 선택
    color_preset_id VARCHAR(50),      -- COLOR_BLACK, COLOR_NATURAL, etc.
    tone_manner JSONB,                -- {"type": "ai_recommended|custom", "value": "세련되고 모던한"}

    -- 생성된 콘텐츠 (JSONB)
    generated_content JSONB,

    -- 편집 히스토리 (JSONB Array)
    edit_history JSONB DEFAULT '[]'::jsonb,

    -- 출력
    output_url VARCHAR(500)
);

-- 템플릿 테이블 (레이아웃 + 고정 폰트만 정의)
CREATE TABLE templates (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(100) NOT NULL,
    category VARCHAR(50) NOT NULL,  -- food, fashion
    width INT NOT NULL,
    height INT NOT NULL,
    thumbnail_url VARCHAR(500),

    -- 레이아웃 구조 정의 (JSONB)
    layout JSONB NOT NULL,

    -- 폰트 정의 (템플릿별 고정)
    typography JSONB NOT NULL,   -- {"font_family": "Pretendard", "font_sizes": {...}}

    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 컬러 프리셋 테이블 (5종)

);
```

### 5.2 주요 API 스키마

```python
# schemas.py

from pydantic import BaseModel
from typing import Optional, List
from datetime import date
from uuid import UUID

# === 프로젝트 ===
class ProjectCreate(BaseModel):
    brand_name: str
    description: str
    category: Optional[str] = None
    event_period_start: Optional[date] = None
    event_period_end: Optional[date] = None
    price_info: Optional[str] = None

class ProjectUpdate(BaseModel):
    template_id: Optional[UUID] = None
    color_preset_id: Optional[str] = None      # 컬러 프리셋 ID
    tone_manner: Optional[dict] = None         # 톤앤매너 선택
    generated_content: Optional[dict] = None
    edit_history: Optional[List[dict]] = None
    status: Optional[str] = None

class Project(BaseModel):
    id: UUID
    status: str
    brand_name: str
    description: str
    category: Optional[str]
    template_id: Optional[UUID]
    generated_content: Optional[dict]
    created_at: str
    updated_at: str

# === 생성 ===
class GenerateRequest(BaseModel):
    project_id: UUID
    template_id: UUID
    color_preset_id: str               # 컬러 프리셋 ID
    tone_manner: str                   # 선택된 톤앤매너

class GenerateResult(BaseModel):
    project_id: UUID
    texts: dict
    images: dict
    generated_at: str

# === 템플릿 (레이아웃 전용) ===
class Template(BaseModel):
    id: UUID
    name: str
    category: str
    width: int
    height: int
    thumbnail_url: Optional[str]
    layout: dict                       # 레이아웃 구조
    typography: dict                   # 폰트 정의 (템플릿별 고정)

# === 컬러 프리셋 ===
class ColorPreset(BaseModel):
    id: str                            # COLOR_BLACK, etc.
    name: str                          # 모던 블랙, etc.
    colors: dict                       # primary, secondary, accent, etc.
    recommended_categories: List[str]

# === 톤앤매너 ===
class ToneRecommendRequest(BaseModel):
    brand_name: str
    description: str
    category: Optional[str] = None

class ToneRecommendation(BaseModel):
    id: int
    keywords: str                      # "세련되고 모던한"
    description: str
    example_copy: str
    fit_score: float

# === 이미지 생성/편집 (Nano Banana Pro) ===
class ImageGenRequest(BaseModel):
    prompt: str
    aspect_ratio: str = "3:4"          # 1:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9
    resolution: int = 2048             # 1024, 2048, 4096 (Pro only)
    use_pro: bool = True               # True: Nano Banana Pro, False: Nano Banana

class ImageEditRequest(BaseModel):
    project_id: UUID
    image_id: UUID                     # 편집할 원본 이미지
    background_prompt: str             # 배경 합성 프롬프트
    style_keywords: Optional[List[str]] = None

class StyleTransferRequest(BaseModel):
    project_id: UUID
    image_id: UUID
    style_prompt: str                  # 적용할 스타일 (톤앤매너)

class CompositeRequest(BaseModel):
    project_id: UUID
    image_ids: List[UUID]              # 합성할 이미지들 (최대 14장)
    layout_prompt: str                 # 레이아웃 지시
```

### 6.2 이미지 생성 AI (Nano Banana Pro)

#### 6.2.1 모델 사양

| 모델 | API 모델 ID | 용도 | 특징 |
|------|------------|------|------|
| **Nano Banana** | `gemini-2.5-flash-image` | 미리보기/빠른 생성 | 속도 최적화, 비용 효율적 |
| **Nano Banana Pro** | `gemini-3-pro-image-preview` | 최종 출력 | 고품질, 4K 지원, 최대 14장 참조 이미지 |

#### 6.2.2 지원 기능

| 기능 | 설명 | POP 생성 활용 |
|------|------|--------------|
| **Text-to-Image** | 텍스트 프롬프트 → 이미지 생성 | 배경, 배너, 데코 요소 생성 |
| **Image Editing** | 이미지 + 텍스트 → 편집된 이미지 | 상품 이미지 배경 합성, 스타일 적용 |
| **Multi-turn Editing** | 대화형 이미지 편집 | 반복적인 수정 작업 |
| **다중 참조 이미지** | 최대 14장 (Pro) | 여러 상품 조합, 일관된 스타일 유지 |

#### 6.2.3 출력 옵션

```
해상도: 1K (기본) | 2K | 4K (Pro만)
종횡비: 1:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9
생성 개수: 1~4장
```

#### 6.2.4 서비스 구현

```python
# services/image_gen_service.py

from google import genai
from google.genai import types
from PIL import Image
from io import BytesIO
from typing import Optional, List
import base64

class ImageGenService:
    """Nano Banana / Nano Banana Pro 이미지 생성 서비스"""

    def __init__(self, api_key: str):
        self.client = genai.Client(api_key=api_key)

        # 모델 선택: 미리보기용 vs 최종 출력용
        self.model_fast = "gemini-2.5-flash-image"      # Nano Banana
        self.model_pro = "gemini-3-pro-image-preview"   # Nano Banana Pro

    async def generate_banner(
        self,
        prompt: str,
        aspect_ratio: str = "3:4",
        resolution: int = 2048,
        use_pro: bool = True
    ) -> bytes:
        """Text-to-Image: 배너 이미지 생성"""

        model = self.model_pro if use_pro else self.model_fast

        response = self.client.models.generate_content(
            model=model,
            contents=[prompt],
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE'],
                output_options=types.OutputOptions(
                    image_config=types.ImageConfig(
                        output_width=resolution,
                        aspect_ratio=aspect_ratio,
                        number_of_images=1
                    )
                )
            )
        )

        # 이미지 추출
        for part in response.candidates[0].content.parts:
            if part.inline_data:
                return base64.b64decode(part.inline_data.data)

        return None

    async def edit_with_background(
        self,
        product_image: bytes,
        background_prompt: str,
        style_keywords: List[str] = None
    ) -> bytes:
        """Image Editing: 상품 이미지 + 배경 합성"""

        # 업로드된 상품 이미지 로드
        img = Image.open(BytesIO(product_image))

        # 스타일 키워드 추가
        full_prompt = background_prompt
        if style_keywords:
            full_prompt += f", {', '.join(style_keywords)}"

        response = self.client.models.generate_content(
            model=self.model_pro,
            contents=[
                full_prompt,
                img  # 업로드된 이미지
            ],
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE'],
                output_options=types.OutputOptions(
                    image_config=types.ImageConfig(
                        output_width=2048,
                        aspect_ratio="3:4"
                    )
                )
            )
        )

        for part in response.candidates[0].content.parts:
            if part.inline_data:
                return base64.b64decode(part.inline_data.data)

        return None

    async def apply_style_transfer(
        self,
        source_image: bytes,
        style_prompt: str
    ) -> bytes:
        """Image Editing: 스타일 전이 (톤앤매너 적용)"""

        img = Image.open(BytesIO(source_image))

        response = self.client.models.generate_content(
            model=self.model_pro,
            contents=[
                f"이 이미지에 다음 스타일을 적용하세요: {style_prompt}",
                img
            ],
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE']
            )
        )

        for part in response.candidates[0].content.parts:
            if part.inline_data:
                return base64.b64decode(part.inline_data.data)

        return None

    async def composite_products(
        self,
        product_images: List[bytes],
        layout_prompt: str
    ) -> bytes:
        """다중 이미지 합성 (최대 14장, Pro만)"""

        images = [Image.open(BytesIO(img)) for img in product_images]

        response = self.client.models.generate_content(
            model=self.model_pro,
            contents=[layout_prompt] + images,
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE'],
                output_options=types.OutputOptions(
                    image_config=types.ImageConfig(
                        output_width=2048,
                        aspect_ratio="3:4"
                    )
                )
            )
        )

        for part in response.candidates[0].content.parts:
            if part.inline_data:
                return base64.b64decode(part.inline_data.data)

        return None

    async def generate_variants(
        self,
        prompt: str,
        count: int = 4
    ) -> List[bytes]:
        """다중 배리에이션 생성 (최대 4장)"""

        response = self.client.models.generate_content(
            model=self.model_pro,
            contents=[prompt],
            config=types.GenerateContentConfig(
                response_modalities=['IMAGE'],
                output_options=types.OutputOptions(
                    image_config=types.ImageConfig(
                        number_of_images=min(count, 4)
                    )
                )
            )
        )

        results = []
        for part in response.candidates[0].content.parts:
            if part.inline_data:
                results.append(base64.b64decode(part.inline_data.data))

        return results
```

### 7.2 편집기 컴포넌트 구조

```tsx
// components/editor/PageEditor.tsx

'use client';

import { useState } from 'react';
import { toPng } from 'html-to-image';
import { useEditorStore } from '@/stores/editorStore';

export function PageEditor({ project, template }) {
  const { content, updateText, updateImage } = useEditorStore();
  const previewRef = useRef<HTMLDivElement>(null);

  // 이미지 출력
  const handleExport = async () => {
    if (!previewRef.current) return;

    const dataUrl = await toPng(previewRef.current, {
      width: template.width,
      height: template.height,
      pixelRatio: 2  // 고화질
    });

    // 다운로드
    const link = document.createElement('a');
    link.download = `${project.brand_name}_${Date.now()}.png`;
    link.href = dataUrl;
    link.click();
  };

  return (
    <div className="flex gap-4">
      {/* 편집 도구 패널 */}
      <aside className="w-64 border-r p-4">
        <TextEditPanel
          content={content.texts}
          onUpdate={updateText}
        />
        <ImageEditPanel
          images={content.images}
          onUpdate={updateImage}
        />
      </aside>

      {/* 미리보기 */}
      <main className="flex-1 p-4">
        <div
          ref={previewRef}
          style={{ width: template.width, height: template.height }}
          className="bg-white shadow-lg"
        >
          <TemplateRenderer
            template={template}
            content={content}
          />
        </div>

        <button onClick={handleExport}>
          PNG 다운로드
        </button>
      </main>
    </div>
  );
}
```
