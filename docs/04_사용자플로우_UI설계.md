## 1. 전체 사용자 플로우 (v4.0 - 동적 레이아웃)

> **핵심 원칙**: 레이아웃 동적 생성 + 텍스트 정확도 확보

```

    ────────                ──────────────             ────────           ────────

  ┌─────────────┐         ┌─────────────────┐       ┌─────────────┐   ┌─────────────┐
  │ 폼 입력     │         │ AI 자동 처리     │       │ 레이아웃 +   │   │  웹 기반    │

        │                         │                       │
```

### 기존 vs 신규 플로우 비교

| 항목 | 기존 (v3.0) | 신규 (v4.0) |
|------|-------------|-------------|
| **핵심 가치** | AI 텍스트 생성 | **동적 레이아웃 생성** |
| **텍스트 처리** | AI가 생성 | **코드로 오버레이** (정확도 확보) |
| **AI 역할** | 카피라이팅 | **비주얼 레이아웃 생성** |
| **출력 형태** | 이미지 파일 | **웹 기반** (HTML/CSS) |
| **엑셀 업로드** | 필수 기능 | Nice-to-have |

## 3. 프론트엔드 컴포넌트 구조

### 3.1 핵심 컴포넌트: TextOverlayRenderer

```tsx
// components/layout/TextOverlayRenderer.tsx

interface TextArea {
  id: number;
  position: string;
  bounds: {
    x: number;      // % 단위
    y: number;      // % 단위
    width: number;  // % 단위
    height: number; // % 단위
  };
  recommended_font_color: string;
  suitable_for: "headline" | "subtext" | "body" | "label";
}

interface Props {
  layoutImageUrl: string;      // Nano Banana Pro 생성 이미지
  textAreas: TextArea[];       // Gemini Vision 추출 좌표
  products: Product[];         // 사용자 입력 상품 정보
  brandName: string;
}

export function TextOverlayRenderer({
  layoutImageUrl,
  textAreas,
  products,
  brandName
}: Props) {
  return (
    <div className="relative w-full aspect-[3/4]">
      {/* 배경: AI 생성 레이아웃 이미지 */}
      <img
        src={layoutImageUrl}
        alt="Layout"
        className="absolute inset-0 w-full h-full object-cover"
      />

      {/* 텍스트 오버레이 레이어 */}
      {textAreas.map((area) => (
        <div
          key={area.id}
          className="absolute flex items-center justify-center text-center"
          style={{
            left: `${area.bounds.x}%`,
            top: `${area.bounds.y}%`,
            width: `${area.bounds.width}%`,
            height: `${area.bounds.height}%`,
            color: area.recommended_font_color,
          }}
        >
          {renderTextContent(area, brandName, products)}
        </div>
      ))}
    </div>
  );
}
```
